# My2DGame 프로젝트 분석 보고서

## 프로젝트 개요
**My2DGame**은 Java Swing을 사용하여 개발된 기본적인 2D 게임 프로젝트입니다. 타일 기반의 화면 시스템과 키보드 입력을 통한 플레이어 움직임 제어 기능을 구현한 간단한 게임 엔진의 기초 구조를 가지고 있습니다.

## 프로젝트 구조

```
My2DGame/
├── bin/                    # 컴파일된 클래스 파일들
│   └── main/
│       ├── Main.class
│       ├── GamePanel.class
│       └── KeyHandler.class
└── src/                    # 소스 코드
    └── main/
        ├── Main.java       # 메인 애플리케이션 진입점
        ├── GamePanel.java  # 게임 화면 및 로직 처리
        └── KeyHandler.java # 키보드 입력 처리
```

## 클래스별 상세 분석

### 1. Main.java (src/main/Main.java:6)
**역할**: 애플리케이션의 진입점
- **주요 기능**:
  - JFrame 윈도우 생성 및 설정
  - 게임 패널 초기화 및 추가
  - 윈도우 속성 설정 (크기 고정, 중앙 배치, 제목 설정)
  - 게임 스레드 시작

### 2. GamePanel.java (src/main/GamePanel.java:10)
**역할**: 게임의 핵심 로직과 화면 렌더링 담당
- **주요 속성**:
  - 화면 설정: 768x576 픽셀 (16x12 타일, 48x48 픽셀 크기)
  - FPS: 60프레임
  - 플레이어 초기 위치: (100, 100)
  - 플레이어 이동 속도: 4픽셀/프레임

- **주요 메서드**:
  - `startGameThread()`: 게임 루프 스레드 시작
  - `run()`: 게임 루프 실행 (Delta Time 방식 사용)
  - `update()`: 게임 상태 업데이트 (플레이어 위치 계산)
  - `paintComponent()`: 화면 렌더링 (흰색 사각형으로 플레이어 표시)

### 3. KeyHandler.java (src/main/KeyHandler.java:6)
**역할**: 키보드 입력 이벤트 처리
- **지원 키**:
  - W: 위쪽 이동
  - A: 왼쪽 이동
  - S: 아래쪽 이동
  - D: 오른쪽 이동
- **구현 방식**: KeyListener 인터페이스 구현

## 기술적 특징

### 게임 루프 구현
- **Delta Time 방식**: 프레임 독립적인 게임 루프 구현 (GamePanel.java:76-105)
- **FPS 모니터링**: 실시간 FPS 출력 기능 (GamePanel.java:100)

### 화면 렌더링
- **더블 버퍼링**: 화면 깜빡임 방지
- **타일 기반 시스템**: 16x16 기본 타일을 3배 확대하여 48x48 픽셀로 사용
- **해상도**: 768x576 픽셀 (16x12 타일)

### 입력 처리
- **즉시 반응**: 키 입력 시 즉시 플레이어 이동
- **연속 입력**: 키를 누르고 있는 동안 지속적인 이동
